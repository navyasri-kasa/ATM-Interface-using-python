# 🏧 Python GUI ATM System

A simple and interactive **ATM Management System** built using **Python** and **Tkinter**. This desktop application simulates the basic functionalities of an Automated Teller Machine (ATM) through an easy-to-use graphical user interface.

## 📌 Features

- 🔐 Secure user login with User ID and PIN
- 💰 Check account balance
- 💵 Deposit money
- 💸 Withdraw money with insufficient balance validation
- 📜 View transaction history
- 🚪 Logout functionality
- 🖥️ Clean and user-friendly GUI built with Tkinter

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI Library)

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your computer

Tkinter comes pre-installed with Python, so no additional libraries are required.

### Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Navigate to the Project Folder

```bash
cd your-repository
```

### Run the Application

```bash
python atm.py
```

## ⚠️ Disclaimer

This project is created for **educational and learning purposes only**. It uses an in-memory mock database, meaning all account information and transaction history are stored temporarily while the program is running. No real banking operations or persistent data storage are involved.
